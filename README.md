# vbwd-plugin-cms-ai

vbwd backend plugin: cms-ai — prompt-driven content & SEO generation.


## Vendored dependency: `loopai`

This plugin depends on a vendored `loopai` tree at `cms-ai/loopai/`, which is **not committed**
to this repository (third-party code, vendored separately). Provision it into `cms-ai/loopai/`
before running the plugin.
